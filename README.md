# Instagram Downloader (Clean Template)

## ✅ Structure
- `index.html` الصفحة الرئيسية
- `info.html` صفحة معلومات / FAQ
- `js/` سكربتات
- `assets/` ملفات الصور/الصوت/الخلفية
- `manifest.json` + `sw.js` لدعم PWA

## تشغيل محلي
افتح `index.html` مباشرة أو استخدم Live Server (مستحسن).

## ملاحظة
هذا قالب جاهز. لو عندك API حقيقي لاستخراج رابط تحميل مباشر:
ضعه في `js/config.js` داخل `apiEndpoint`.
